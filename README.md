# README - Shady Pines Murder Mystery

Welcome to **"The 40-Year-Old Retirement"** - a hilarious murder mystery dinner party for 12-20 guests celebrating a 40th birthday with retirement home chaos!

## 🎭 Quick Overview

- **Setting**: Shady Pines Retirement Community
- **Event**: Bingo Championship Night (and a 40th birthday celebration)
- **The Twist**: A murder occurs during BINGO!
- **Theme**: Retirement home humor with Golden Girls-inspired characters

## 🎬 How to Run This Party

### Before the Party
1. Read `host/timeline.md` - your master guide
2. Choose the victim (resident or staff member)
3. Choose the murder method (see `story/the_murder.md`)
4. Decide who the killer is (or let it emerge naturally)
5. Print character bios from `characters/` folder
6. Prepare props (see props list in `host/objectives_and_quirks.md`)
7. Set up "locations" with signs

### During the Party
Follow the phases in `host/timeline.md`:
1. **Arrivals** - Character distribution and mingling
2. **Bingo Begins** - Natural clue collection
3. **THE MURDER!** - Big acting spurt
4. **Investigation** - Players solve the mystery
5. **Accusation** - The reveal!
6. **Celebration** - Back to birthday party!

### If You Need Help
- **Players stuck?** See "If Players Get Stuck" in `host/timeline.md`
- **Too fast/slow?** Timing adjustments in timeline
- **Need more clues?** Check `clues/database.md`

## 🎨 Customization Options

### Scaling for Different Group Sizes

**12 people** (minimum):
- 8 primary characters + 4 NPCs = Perfect!

**14-16 people**:
- Add 2-4 more residents from "Expansion Options" in `characters/manifest.md`
- Suggestions: Bernie (Dorothy's boyfriend), Sal (her other boyfriend), The Veteran, The Artist

**18-20 people**:
- Add additional staff (Cook, Janitor, PT, Administrator)
- OR give some primary characters "assistants" who share clues

### Adjusting Difficulty

**Easier**:
- Make NPCs give clues more readily
- Provide written clue summary sheets
- Host announces "hot" vs "cold" theories

**Harder**:
- More red herrings
- Conflicting clues (memory problems!)
- Multiple possible killers (decide based on evidence)
