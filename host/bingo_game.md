# Bingo Mini-Game System

Turn bingo night into an actual playable game that integrates with the murder mystery!

---

## 🎲 Bingo Game Overview

### Core Concept
- **Host calls bingo numbers** during Phase 2 (Bingo Night)
- **Players mark their cards** with daubers/pens
- **First to complete a pattern** gets bonus points AND a special clue
- **Multiple rounds** = multiple winners = more clues revealed

### Integration with Murder Mystery
- Each bingo card has a **hidden clue** on the back
- When someone wins, they **flip the card over** and read the clue aloud
- Clues are randomized so different people get different information
- Winning bingo = **2 bonus points** + the clue (worth 0.5 pts as usual)

---

## 🎯 Bingo Rules & Patterns

### Round 1: Traditional "Five in a Row"
- Horizontal, vertical, or diagonal line
- First person to complete shouts "BINGO!"
- Winner gets **2 pts** + **Clue Card #1**

### Round 2: "Four Corners"
- Mark all four corner squares (B1, O75, etc.)
- Faster round, more chaos!
- Winner gets **2 pts** + **Clue Card #2**

### Round 3: "Blackout" (Optional/Time Permitting)
- Fill entire card
- Takes longer, use if you have time
- Winner gets **3 pts** + **Premium Clue Card**

### Pattern Variations (Pick 1-2)
- **X Pattern**: Diagonal lines forming an X
- **Plus Sign**: Middle row + middle column
- **Picture Frame**: All outer edges
- **Letter "H"**: Both outer columns + middle row

## 📊 Bingo Scoring Integration

### Points for Winning Bingo
- **Round 1 (Five in a Row)**: 2 points + clue reveal
- **Round 2 (Four Corners)**: 2 points + clue reveal
- **Round 3 (Blackout)**: 3 points + premium clue
- **Additional Bingo** (same round): 1 point each

### How It Works with Overall Scoring
- Bingo wins are **separate from** regular clue discovery
- Bingo clues still count as 0.5 pts each (when written on tracking sheet)
- So winning Round 1 = **2 pts (bingo win) + 0.5 pts (clue)** = 2.5 total
- Multiple people can win the same round (tie = both get points)

### Updated Scoring Sheet Section
Add to player tracking sheet:

```
BINGO WINS:
□ Round 1 (Five in a Row) - 2 pts
□ Round 2 (Four Corners) - 2 pts  
□ Round 3 (Blackout) - 3 pts
□ Additional Win - 1 pt

BINGO TOTAL: _____ pts
```

## 🎮 Host Guide: Running Bingo

### Setup (Before Party)
1. **Print 12 bingo cards** (double-sided with clues on back)
2. **Prepare bingo calling equipment**:
   - Bingo ball cage OR number generator app
   - Calling sheet to track numbers called
3. **Distribute cards** when guests arrive (one per person)
4. **Provide daubers/pens** for marking

### During Phase 2: Bingo Night

**Round 1: Five in a Row (10-15 minutes)**

Host (as Gerald): 
> "Alright my groovy cats! Time for CHAMPIONSHIP BINGO! First one to get five in a row—horizontal, vertical, or diagonal—shouts BINGO! Let's get radical!"

1. Call numbers at moderate pace (every 15-20 seconds)
2. Players mark their cards
3. First "BINGO!" → Verify the win
4. Winner flips card and **reads clue aloud**
5. Winner records: +2 pts for bingo win, +0.5 pts for clue
6. Continue until 2-3 people win (or time limit)

**Interlude: Conversation & Clue Sharing (5 minutes)**
- Players discuss the revealed clues
- Trade information, use tokens
- Rose runs her gambling ring between rounds

**Round 2: Four Corners (5-7 minutes)**

Host:
> "Round 2, baby! Four corners only—fast and furious! Who's got the luck?!"

1. Reset cards OR use same cards (different pattern)
2. Call numbers faster
3. First to mark all four corners wins
4. Winner reveals their clue card
5. Award points

**Optional Round 3: Blackout (if time allows)**
- Full card coverage
- Takes 15-20 minutes
- Premium clue worth more murder-solving value

### If Multiple Winners (Same Number)
- Both get full points!
- Both reveal their clue cards
- More clues in circulation = better investigation

### If Someone Already Won Their Card
- They can still play for 1 bonus point
- OR give them a second card to play

## 🎯 Bingo Strategy Tips for Players

Share these tips during setup:

1. **Listen carefully** - Don't miss called numbers!
2. **Mark clearly** - Use daubers or thick pen
3. **Watch others** - If everyone's close, work faster
4. **FREE space** counts for all patterns
5. **Shout loudly** - First person heard wins ties
6. **Read clues dramatically** - It's more fun!
